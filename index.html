Create your own custom category

To create your own custom category of persons, follow the steps below. If you build a custom category you think others would enjoy, email it to me, and I'll consider adding it to the menu. Data sets must not exceed 100 people.

  1. Copy the contents of the JSON file linked below. Using FamilySearch replace the PIDs and data with your own. You can verify your JSON file is syntactically correct by using JSON Formatter.

    https://cousin.surge.sh/salem.json

  2. Upload your new JSON file to the Internet somewhere that is accessible. npoint.io is a free place to host JSON files.

  3. To use your new data set, put the full URL of your new JSON file in the URL of this website like the example below.

    https://cousin.surge.sh/rel.html?set=https://example.com/custom.json

    (You can embed the URL above in an iframe on your website.)
